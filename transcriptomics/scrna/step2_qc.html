QC and Filtering

Step 02 / 08

scRNA-seq 데이터 분석의 첫 번째 핵심 단계는 품질 관리(QC)입니다. 저품질 세포, 손상된 세포, 잠재적 doublet을 점검하고 제거하여 이후 정규화, 차원축소, 클러스터링 결과의 신뢰도를 높입니다.

A) 왜 QC가 중요한가?

  • scRNA-seq 데이터는 bulk RNA-seq보다 sparse하며 technical noise가 큽니다.
  • 손상되거나 죽어가는 세포는 mitochondrial gene 비율 증가, 낮은 유전자 수 등의 패턴을 보일 수 있습니다.
  • QC를 제대로 하지 않으면 이후 클러스터링에서 가짜 cluster가 생기거나, 특정 세포군 해석이 왜곡될 수 있습니다.
  • threshold는 고정값이라기보다, 데이터 분포를 보고 합리적으로 설정하는 것이 중요합니다.
핵심 메시지
  • QC는 “지저분한 세포 제거”가 아니라, 분석 가능한 세포를 선별하는 과정입니다.
  • 너무 엄격한 기준은 실제 biological signal을 잃게 할 수 있고, 너무 느슨한 기준은 noise를 남길 수 있습니다.

B) 주요 QC 지표

지표 의미 및 해석
nFeature_RNA 세포에서 검출된 유전자 수. 너무 낮으면 저품질 세포, 너무 높으면 doublet 가능성을 의심할 수 있습니다.
nCount_RNA 세포의 총 UMI count. 세포 내 RNA 양과 library size를 반영하지만, 지나치게 높으면 doublet 가능성이 있습니다.
percent.mt 미토콘드리아 유전자 비율. 일반적으로 높을수록 손상되거나 dying cell 가능성이 커집니다.

D) QC 지표 시각화

보통 violin plot과 scatter plot을 함께 사용하여 데이터 분포를 확인합니다. threshold를 정하기 전에 반드시 분포를 먼저 확인하는 것이 좋습니다.

D-1) Violin plot

VlnPlot(
  obj,
  features = c("nFeature_RNA", "nCount_RNA", "percent.mt"),
  ncol = 3,
  pt.size = 0.1
)
D-2) Scatter plot

FeatureScatter(obj, feature1 = "nCount_RNA", feature2 = "percent.mt")
FeatureScatter(obj, feature1 = "nCount_RNA", feature2 = "nFeature_RNA")

QC 단계에서는 nFeature_RNA, nCount_RNA, percent.mt의 분포를 확인하여 저품질 세포와 이상치를 점검합니다.

E) Filtering 기준 설정 예시

아래는 예시일 뿐이며, 데이터셋에 따라 cut-off는 달라질 수 있습니다. 조직 종류, dissociation 상태, species, chemistry version에 따라 조정이 필요합니다. 


obj <- subset(
  obj,
  subset =
    nFeature_RNA > 200 &
    nFeature_RNA < 6000 &
    percent.mt < 10
)

obj
threshold 설정 팁
  • nFeature_RNA < 200 미만 세포는 보통 저품질 세포로 간주하는 경우가 많습니다.
  • percent.mt cut-off는 5%, 10%, 15% 등 데이터에 따라 달라질 수 있습니다.
  • 상한선(nFeature_RNA, nCount_RNA)은 doublet 또는 multiplet 가능성을 고려해 설정합니다.

F) Filtering 전후 비교

filtering 후에는 세포 수와 QC 지표 분포가 어떻게 달라졌는지 다시 확인합니다.

G) Doublet에 대한 간단한 이해

  • doublet은 하나의 barcode에 두 개 이상의 세포가 함께 잡힌 경우입니다.
  • 이 경우 비정상적으로 높은 nFeature_RNA, 높은 nCount_RNA를 보일 수 있습니다.
  • 기본 QC에서 일부 의심 세포를 줄일 수 있지만, 본격적인 검출은 이후 DoubletFinder 같은 도구를 사용할 수 있습니다.

H) 실습용 예시 코드 정리


# 1. mitochondrial ratio 계산
obj[["percent.mt"]] <- PercentageFeatureSet(obj, pattern = "^MT-")

# 2. QC plot 확인
VlnPlot(obj, features = c("nFeature_RNA", "nCount_RNA", "percent.mt"), ncol = 3)
FeatureScatter(obj, feature1 = "nCount_RNA", feature2 = "percent.mt")
FeatureScatter(obj, feature1 = "nCount_RNA", feature2 = "nFeature_RNA")

# 3. filtering
obj <- subset(
  obj,
  subset =
    nFeature_RNA > 200 &
    nFeature_RNA < 6000 &
    percent.mt < 10
)

# 4. filtering 후 재확인
VlnPlot(obj, features = c("nFeature_RNA", "nCount_RNA", "percent.mt"), ncol = 3)
dim(obj)
요약
  • QC는 이후 분석 결과의 신뢰도를 결정하는 매우 중요한 단계입니다.
  • 주요 지표는 nFeature_RNA, nCount_RNA, percent.mt입니다.
  • threshold는 고정값이 아니라, 데이터 분포를 먼저 보고 설정해야 합니다.
  • filtering 후에는 반드시 세포 수와 분포를 다시 점검해야 합니다.
