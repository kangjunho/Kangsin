Introduction to scRNA-seq and R Server Environment

Step 01 / 08

single-cell RNA-seq의 기본 개념을 이해하고, 웹 기반 RStudio Server를 통해 분석 환경에 접속한 뒤 NAS 데이터 저장소를 활용하여 실습을 시작합니다.

A) scRNA-seq 개요

  • scRNA-seq는 개별 세포 단위의 유전자 발현을 분석하는 기술입니다.
  • bulk RNA-seq는 평균값을 분석하지만, scRNA-seq는 세포 간 이질성을 분석합니다.

B) Bulk vs Single-cell

항목BulkSingle-cell
단위샘플 전체개별 세포
특징평균 발현이질성 분석
결과gene × samplegene × cell

C) 핵심 개념

  • Barcode: 세포 식별
  • UMI: PCR 중복 제거
  • Count matrix: gene × cell

D) 10X 데이터 구조


filtered_feature_bc_matrix/
├── barcodes.tsv.gz
├── features.tsv.gz
└── matrix.mtx.gz

E) RStudio Server 접속

1) 웹 접속

http://server_ip:8787

브라우저에서 접속하면 RStudio Server 로그인 화면이 나타납니다. 제공된 ID와 Password를 입력하여 로그인합니다.

2) 로그인 후 환경
  • Source: 코드 작성
  • Console: 실행
  • Environment: 객체 확인
  • Files: 파일 탐색

F) NAS 데이터 접근

분석 환경에 접속하면, 사용자는 반드시 개인 데이터 저장소(NAS)에 접근해야 합니다. 실습 데이터 및 분석 결과는 NAS 경로에 저장됩니다. 


/nas/username/scRNA_project/
중요
  • 데이터는 반드시 NAS 경로에서 불러와야 합니다.
  • 로컬 경로와 혼동하지 않도록 주의합니다.
  • 용량이 큰 데이터는 서버/NAS에서 직접 처리합니다.

H) Seurat 시작


library(Seurat)

data_dir <- "/nas/username/scRNA_project/sample1/filtered_feature_bc_matrix"

counts <- Read10X(data.dir = data_dir)

obj <- CreateSeuratObject(counts = counts)
