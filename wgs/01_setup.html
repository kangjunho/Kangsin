Step 01. Environment Setup

분석을 시작하기 전에 Conda 환경과 필수 툴을 설치하고, 레퍼런스 파일을 준비합니다. 앞으로 모든 분석은 20 threads를 기본으로 사용합니다.

1) Conda 환경 생성


# 채널 추가 (처음 한 번만 실행)
conda config --add channels conda-forge
conda config --add channels bioconda
conda config --set channel_priority strict

# 환경 생성
conda create -y -n wgs_env python=3.10 \
  cutadapt bwa samtools gatk4 bcftools bedtools fastqc multiqc \
  openjdk=17 parallel

# 환경 활성화
conda activate wgs_env

2) 디렉토리 구조 예시


/home/kang/
 ├── raw_data/               # 원본 데이터 및 reference
 │    └── gatk_bundle/       # reference genome + vcf files
 ├── God_Nas/WGS_2nd/        # 프로젝트 폴더
 │    └── Trimmed_fastq/     # 트리밍된 FASTQ 저장
 └── ...

3) Reference Genome 준비

분석에는 Homo_sapiens_assembly38.fasta (hg38)를 사용합니다.


# 환경 변수 지정
REF_DIR=/home/kang/raw_data/gatk_bundle
REF=${REF_DIR}/Homo_sapiens_assembly38.fasta

# 인덱스 생성
bwa index ${REF}
samtools faidx ${REF}
gatk CreateSequenceDictionary -R ${REF} -O ${REF%.fasta}.dict

4) Thread 설정


# 앞으로 모든 스크립트 상단에 추가
THREADS=20
