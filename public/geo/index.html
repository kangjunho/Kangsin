GEO Microarray Analysis

Windows 환경 설정부터 GEO 공개데이터의 DEG 및 기능분석까지 진행하는 입문 실습입니다.

초급 · R 기반 · 단계별 실습

교육 목표

GEO에서 연구 목적에 맞는 microarray 데이터를 찾고, R을 이용해 재현 가능한 분석을 수행합니다.

하나의 예제 데이터셋을 이용하여 다운로드, 임상정보 정리, QC, DEG, 시각화 및 기능분석을 순서대로 연결합니다.

실습 과정

Step 00부터 순서대로 진행하세요.

00

R과 RStudio 설치

Windows에서 R, RStudio 및 실습 패키지를 설치하고 환경을 점검합니다.

실습 시작
01

R 기초와 작업환경

RStudio 화면, 프로젝트, 작업 폴더, 패키지와 데이터프레임을 익힙니다.

실습 시작
02

GEO 데이터 검색

GEO의 GSE, GSM, GPL 구조를 이해하고 적절한 데이터셋을 선택합니다.

실습 시작
03

데이터셋 선정

GSE record와 metadata를 읽고 분석 가능한 비교를 정의합니다.

실습 시작
04

GEO2R 빠른 분석

웹에서 그룹을 정의하고 DEG 결과와 분석 방향을 확인합니다.

실습 시작
05

GEO2R R script

웹 결과의 R 코드를 확인하고 재현 가능한 분석으로 연결합니다.

실습 시작
06

GEOquery 다운로드

발현행렬, sample metadata와 probe annotation을 R로 가져옵니다.

실습 시작
07

전처리와 QC

분포, PCA, 상관성과 probe annotation을 점검합니다.

실습 시작
08

limma DEG

Design과 contrast를 구성하고 차등발현 유전자를 선별합니다.

실습 시작
09

결과 시각화

PCA, volcano, heatmap과 결과표를 작성하고 저장합니다.

실습 시작
10

DAVID GO·Pathway

Up·Down DEG에서 과대표현된 기능과 경로를 탐색합니다.

실습 시작
11

STRING network

단백질 연관 network와 PPI enrichment를 해석합니다.

실습 시작
12

GSEA·MSigDB

전체 유전자 순위에서 일관된 pathway 변화를 찾습니다.

실습 시작
13

결과 통합과 보고

분석 결과, 한계, 버전과 재현성 정보를 정리합니다.
