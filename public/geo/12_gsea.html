GSEA·MSigDB 공식 사이트

유전자 순위 만들기

# 권장 예: moderated t-statistic
gene_rank <- deg_result$t
names(gene_rank) <- deg_result$gene
gene_rank <- gene_rank[!is.na(gene_rank)]
gene_rank <- sort(gene_rank, decreasing = TRUE)
중복 gene symbol은 하나의 대표값으로 정리해야 합니다. P-value만으로 순위를 만들면 방향성이 사라질 수 있습니다.

MSigDB collection

  • Hallmark: 중복을 줄인 핵심 생물학적 프로그램
  • GO Biological Process: 세부 기능과 과정
  • Reactome/curated pathways: 정리된 신호·대사경로
  • Human/Mouse: 종에 맞는 collection 선택

R에서 실행 예시

library(msigdbr)
library(fgsea)

msig_h <- msigdbr(
  species = "Homo sapiens",
  collection = "H"
)
pathways <- split(msig_h$gene_symbol, msig_h$gs_name)

gsea_result <- fgsea(
  pathways = pathways,
  stats = gene_rank,
  minSize = 15,
  maxSize = 500
)
gsea_result <- gsea_result[order(gsea_result$padj), ]

# CSV 저장 전 list 열을 문자로 변환
gsea_result$leadingEdge <- vapply(
  gsea_result$leadingEdge, paste, collapse = ";",
  FUN.VALUE = character(1)
)

결과 해석

  • NES > 0: 순위 상단, 즉 Test 방향에 집중
  • NES < 0: 순위 하단, 즉 Control 방향에 집중
  • FDR: 여러 gene set 검사에 대한 보정 결과
  • Leading edge: enrichment signal에 크게 기여한 핵심 유전자
경로 이름만 나열하지 말고 NES 방향, FDR와 leading-edge gene을 DEG·STRING 결과와 연결합니다.
