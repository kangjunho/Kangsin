Step 07

전처리, QC와 probe annotation

분포·표본 관계·annotation을 점검하여 분석 가능한 gene matrix를 만듭니다.

log2 변환 판단

qx <- quantile(expr_matrix, c(0, .25, .5, .75, .99, 1), na.rm = TRUE)
log2_needed <- qx[5] > 100 || (qx[6] - qx[1] > 50)

if (log2_needed) {
  expr_matrix[expr_matrix <= 0] <- NA
  expr_matrix <- log2(expr_matrix)
}
수치 범위만으로 log2 여부를 확정하지 않습니다. GEO record, value distribution과 제공자의 처리방법을 함께 확인합니다.

기본 QC

boxplot(expr_matrix, outline = FALSE, las = 2,
        main = "Expression distribution")

sample_cor <- cor(expr_matrix, use = "pairwise.complete.obs")
pheatmap::pheatmap(sample_cor)

pca <- prcomp(t(expr_matrix), scale. = FALSE)
plot(pca$x[, 1], pca$x[, 2], pch = 19)

Boxplot은 분포 차이, correlation과 PCA는 outlier·그룹·batch 구조를 확인하는 데 사용합니다.

Probe에서 gene으로

colnames(feature_info)

# 실제 symbol 열 이름으로 변경
probe_map <- data.frame(
  probe_id = rownames(feature_info),
  gene_symbol = feature_info$`Gene Symbol`
)

probe_map <- probe_map[
  !is.na(probe_map$gene_symbol) & probe_map$gene_symbol != "",
]
모든 플랫폼에서 org.Hs.eg.db의 PROBEID를 직접 사용할 수 있는 것은 아닙니다. GPL annotation 또는 해당 platform annotation package를 사용합니다.

중복 probe 병합

한 gene에 여러 probe가 대응하면 평균, 중앙값 또는 가장 큰 변이를 가진 probe 등 사전에 정한 기준으로 병합합니다.

# limma::avereps 사용 예시
gene_matrix <- limma::avereps(
  expr_matrix[probe_map$probe_id, ],
  ID = probe_map$gene_symbol
)
