limma 차등발현 분석

분석 방향을 명시한 design과 contrast로 Disease와 Control을 비교합니다.

그룹과 design matrix

group <- factor(
  sample_info$group,
  levels = c("Control", "Disease")
)

design <- model.matrix(~ 0 + group)
colnames(design) <- levels(group)
table(group)
design
예제의 sample_info$group은 실제 metadata를 확인한 뒤 직접 만들어야 합니다.

Contrast와 limma

contrast_matrix <- limma::makeContrasts(
  Disease_vs_Control = Disease - Control,
  levels = design
)

fit <- limma::lmFit(gene_matrix, design)
fit <- limma::contrasts.fit(fit, contrast_matrix)
fit <- limma::eBayes(fit)

deg_result <- limma::topTable(
  fit,
  coef = "Disease_vs_Control",
  number = Inf,
  adjust.method = "BH",
  sort.by = "P"
)
logFC > 0 : Disease에서 증가
logFC < 0 : Disease에서 감소

DEG 기준

deg_result$status <- "Not significant"
deg_result$status[
  deg_result$adj.P.Val < 0.05 & deg_result$logFC >= 1
] <- "Up"
deg_result$status[
  deg_result$adj.P.Val < 0.05 & deg_result$logFC <= -1
] <- "Down"

table(deg_result$status)

임계값은 연구 목적, 표본 수, 플랫폼과 effect size를 고려하여 분석 전에 정하고 보고서에 명시합니다.
