Step 09

PCA, Volcano와 Heatmap

분석 품질과 차등발현 결과를 이해하기 쉬운 그림과 표로 저장합니다.

PCA

pca <- prcomp(t(gene_matrix), scale. = FALSE)
pca_df <- data.frame(
  PC1 = pca$x[, 1], PC2 = pca$x[, 2],
  group = group
)

ggplot2::ggplot(pca_df, ggplot2::aes(PC1, PC2, color = group)) +
  ggplot2::geom_point(size = 3) +
  ggplot2::theme_classic()

PCA 분리는 생물학적 그룹뿐 아니라 batch, 조직과 outlier에 의해 생길 수 있습니다.

Volcano plot

deg_result$gene <- rownames(deg_result)

p_volcano <- ggplot2::ggplot(
  deg_result,
  ggplot2::aes(logFC, -log10(adj.P.Val), color = status)
) +
  ggplot2::geom_point(alpha = 0.65) +
  ggplot2::scale_color_manual(values = c(
    Down = "#2166AC", `Not significant` = "grey75", Up = "#B2182B"
  )) + ggplot2::theme_classic()

Top DEG heatmap

top_genes <- head(rownames(deg_result), 50)
heatmap_matrix <- gene_matrix[top_genes, , drop = FALSE]

pheatmap::pheatmap(
  heatmap_matrix,
  scale = "row",
  annotation_col = data.frame(Group = group)
)
Heatmap의 row scaling은 유전자별 상대 발현 패턴을 보여주며 유전자 간 절대 발현량 비교에는 사용하지 않습니다.

결과 저장

ggplot2::ggsave("figures/volcano.pdf", p_volcano,
                width = 7, height = 6)
