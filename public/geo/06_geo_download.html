Step 06

GEOquery로 데이터 다운로드

GSE의 발현행렬, sample metadata와 probe annotation을 R 객체로 가져옵니다.

데이터 불러오기

library(GEOquery)

gset <- getGEO(
  "GSE_ACCESSION",
  GSEMatrix = TRUE,
  AnnotGPL = FALSE
)

length(gset)
eset <- gset[[1]]
여러 플랫폼이 사용된 GSE는 gset[[1]], gset[[2]]가 서로 다른 GPL일 수 있습니다. 분석 전에 각 객체의 플랫폼과 표본 수를 확인합니다.

ExpressionSet 구성

expr_matrix  <- exprs(eset)  # probe × sample 발현행렬
sample_info  <- pData(eset)  # sample metadata
feature_info <- fData(eset)  # probe annotation

dim(expr_matrix)
head(sample_info)
head(feature_info)
annotation(eset)
exprs()→발현값pData()→샘플정보fData()→probe정보

metadata 확인

colnames(sample_info)
sample_info[, c("geo_accession", "title", "source_name_ch1")]

# characteristics 열 탐색
grep(
  "characteristics",
  colnames(sample_info),
  value = TRUE
)

그룹을 자동 추측하지 말고 GSE record와 원 논문을 대조해 직접 정의합니다.

중간파일 저장

saveRDS(eset, "data_processed/expression_set.rds")
write.csv(sample_info, "data_processed/sample_metadata.csv",
          row.names = FALSE, fileEncoding = "UTF-8")
