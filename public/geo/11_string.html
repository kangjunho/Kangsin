Step 11

STRING 단백질 연관 네트워크

DEG 단백질의 기능적·물리적 연관성과 network enrichment를 탐색합니다.

STRING이 보여주는 관계

STRING은 알려졌거나 예측된 protein association을 통합합니다. 기본 full network는 직접 결합뿐 아니라 같은 과정에 관여하는 기능적 연관성도 포함합니다.

PPI ≠ 항상 직접 결합: physical subnetwork를 선택해도 직접 binding뿐 아니라 같은 complex 내 간접적 근접성을 포함할 수 있습니다.
STRING 공식 홈페이지

웹 실습 순서

  1. Multiple proteins를 선택합니다.
  2. Up 또는 Down gene symbol을 입력합니다.
  3. Organism을 Homo sapiens로 설정합니다.
  4. full STRING network와 physical subnetwork를 비교합니다.
  5. confidence score를 설정하고 network를 생성합니다.
  6. Analysis에서 PPI enrichment와 functional enrichment를 확인합니다.
  7. Exports에서 network image, TSV와 node degree를 저장합니다.

Network 해석

요소해석
Node단백질을 대표하며 isoform은 보통 locus 단위로 통합
Edge물리적 또는 기능적 association
Edge thicknessconfidence가 높을수록 굵음
Evidence colorexperiment, database, coexpression, text mining 등 근거 유형
PPI enrichment P-value입력 단백질 사이 연결이 무작위 기대보다 많은지 평가
Network 그림의 물리적 거리 자체에는 생물학적 의미가 없습니다.

분석 기록

organism: Homo sapiens (9606)
network type: full / physical
required score: 0.4 or 0.7
additional interactors: 0
input genes: Up or Down DEG
STRING version and analysis date

후보 hub를 선정할 때 degree만 사용하지 말고 network 조건, 발현 방향, pathway와 기존 문헌을 함께 검토합니다.
